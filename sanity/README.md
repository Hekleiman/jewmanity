# Jewmanity — Sanity Studio

Content management system for jewmanity.org.

## Local Development

```bash
cd sanity
npm install
npm run dev
# Opens at http://localhost:3333
```

## Deploy Hosted Studio

```bash
cd sanity
npx sanity deploy
# Choose a hostname like "jewmanity" → available at jewmanity.sanity.studio
```

## Content Types

| Type             | Description                              |
| ---------------- | ---------------------------------------- |
| Recipe           | Community recipes with ingredients/steps |
| Retreat          | Past retreat events                      |
| Team Member      | Staff and board members                  |
| Product          | Shop merchandise                         |
| Testimonial      | Community testimonials                   |
| FAQ Item         | Frequently asked questions               |
| Community Story  | Community impact stories                 |
| Site Settings    | Global site configuration                |
