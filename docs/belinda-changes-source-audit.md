# Belinda's Change List ‚Äî Source-of-Truth Audit

Read-only audit performed 2026-04-20 against branch `main` (HEAD `8c94e84`) + uncommitted working tree.

Goal: for each requested change, identify where the source of truth lives so work can be split between (a) Belinda editing Sanity Studio, (b) CC making code changes, (c) a bulk script run with write token, or (d) both.

Every CMS-driven page uses a `try/catch` fallback pattern: GROQ query ‚Üí if it returns data, use it; else use the hardcoded fallback embedded in the component. "CMS wins at runtime" means whatever is published in Sanity shows up live, and the hardcoded version only renders when the CMS fetch fails OR when the corresponding CMS field is empty/absent (components fall through to fallbacks field-by-field).
