# Jewmanity

A 501(c)(3) nonprofit website supporting mental health and healing for Jewish and Israeli communities through retreats, peer support, education, and ongoing care.

## Stack

- [Astro 5](https://astro.build) — static site framework
- [Tailwind CSS 4](https://tailwindcss.com) — utility-first CSS
- [GSAP](https://gsap.com) — animations
- [Sanity](https://sanity.io) — headless CMS (Phase 3)
- TypeScript — strict mode

## Development

```bash
npm install
npm run dev
```

Open [localhost:4321](http://localhost:4321) in your browser.
